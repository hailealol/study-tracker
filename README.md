This study tracker app contains the following features:
  1. Profile editor to customize your experience (saves when you exit the app)
  2. Events tracker: keep track of upcoming tests by displaying the subject, date, and any personal notes
  3. A timer that notifies you when your studying session is over

Build & Run Steps:
  1. Clone this repo or download the project zip
  2. Open the project in Android Studio
     > Open Android Studio > Click Open an exisitng project & navigate to your download
  3. Click Sync Now when prompted by Android Studio to sync the Gradle
  4. Build the project by going to Build > Assemble Project
  5. Connect desired device (this project was built using the Pixel 9 emulator)
     > Ensure the device you pick supports API 26+ as this project uses features like NotificationChannel
  7. Select your target device from the top device menu and click the run button (green triangle)
